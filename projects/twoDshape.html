2D Shape Detection and Answering Word Problems

  • Developed a geometric shape detection system integrated with a large language model (LLM) to solve shape-based word problems.
  • Designed and implemented robust algorithms to detect and classify both single and nested geometric shapes using computer vision (OpenCV).
  • Enabled inscribed shape identification and shape hierarchy analysis with optimized contour processing techniques.
  • Generated dynamic question-and-answer solutions using the Phi-3 LLM to address geometry-related queries.
  • Built a user-friendly frontend using Gradio for seamless interaction between user input and backend processing.
  • Handled real-time image input and LLM response rendering for enhanced interactivity and educational use cases.

Tools Used: Python, OpenCV, Gradio, Phi-3 LLM
