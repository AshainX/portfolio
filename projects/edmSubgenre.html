EDM Subgenre Classification

  • Built a music genre classification model using a hybrid InceptionV3 + ResNet architecture.
  • Curated a custom dataset of 10 EDM subgenres (e.g., Bigroom, Trance) with audio preprocessing using PyAudioAnalysis and Librosa.
  • Conducted performance comparison across ML models: Random Forest (74.88%), SVM (68.26%), and Logistic Regression (66.20%).
  • Achieved highest accuracy of 81% with a custom CNN, outperforming traditional ML models.

Tools Used: Python, TensorFlow, Librosa, PyAudioAnalysis, Scikit-learn
