Early Dengue Detection using CBC Data

This project implements a machine learning pipeline to predict early signs of dengue using Complete Blood Count (CBC) reports. The system is built in MATLAB and applies preprocessing, data balancing, and classification to deliver accurate predictions from clinical blood data.
