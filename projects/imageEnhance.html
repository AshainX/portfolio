Image Enhancement using Deep Learning

A deep learning-based system that enhances low-light, noisy images into high-quality, well-lit images using a custom Convolutional Neural Network (CNN) built with Keras and TensorFlow.

Functionality

  • Preprocessing of high-light images to generate low-light training data
  • Training a CNN to learn enhancement from noisy to clear images
  • Evaluation with quality metrics (PSNR & SSIM)
  • Side-by-side visualization of original, degraded, and enhanced images

Model Highlights

  • Custom CNN with multi-branch Conv2D architecture
  • Input & output: RGB images (500√ó500)
  • Uses ReLU activations, MSE loss, and Adam optimizer
